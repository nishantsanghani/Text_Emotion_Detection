Text Emotion Detection System (NLP-Based)
========================================

Text Emotion Detection is a Natural Language Processing (NLP) based Machine Learning web application that analyzes user-input text and predicts the underlying emotion along with confidence scores and probability visualization.

The application is built using Streamlit and a pre-trained NLP machine learning model.

--------------------------------------------------
PROJECT FEATURES
--------------------------------------------------
- NLP-based emotion detection from text
- Supports multiple emotions: anger, disgust, fear, joy, neutral, sadness, shame, surprise
- Displays predicted emotion with emoji
- Shows confidence score of prediction
- Visualizes emotion-wise probabilities using bar chart
- Simple, interactive, and user-friendly interface
- No coding required for end users

--------------------------------------------------
TECHNOLOGIES USED
--------------------------------------------------
- Python
- Natural Language Processing (NLP)
- Machine Learning
- Streamlit (Web Application Framework)
- Scikit-learn (ML & NLP Pipeline)
- Pandas & NumPy (Data Handling)
- Altair (Data Visualization)
- Joblib (Model Serialization)

--------------------------------------------------
APPLICATION FLOW (EASY TO UNDERSTAND)
--------------------------------------------------
1. User enters text in the input box
2. Clicks the "Submit" button
3. System processes text using NLP techniques
4. Text is converted into numerical features
5. Machine learning model predicts the emotion
6. Predicted emotion and emoji are displayed
7. Confidence score is shown
8. Emotion probability graph is visualized

--------------------------------------------------
NLP USED IN THIS PROJECT
--------------------------------------------------
- Text preprocessing
- Feature extraction using NLP techniques
- Text vectorization (TF-IDF / Bag of Words)
- Emotion classification using Machine Learning

--------------------------------------------------
HOW TO INSTALL & RUN
--------------------------------------------------
1. Clone the repository
   git clone https://github.com/nishantsanghani/Text_Emotion_Detection

2. Navigate to the project directory
   cd text-emotion-detection

3. Install required dependencies
   pip install -r requirements.txt

4. Run the Streamlit application
   streamlit run app.py
